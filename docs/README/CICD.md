# Sphinx documentation of toulbar2 -> CI/CD

## Install

- The file .github/workflows/docs-deploy.yml launches file docs/_cicd/deploy.sh
- On the repository :
  "Settings" / "Pages" choose 'gh-pages branch' under "Source"
